# Freight Forwarding App

Enterprise-grade Freight Forwarding add-on for ERPNext v15.

## Installation

```bash
bench get-app freight_forwarding <repository-url>
bench --site [site-name] install-app freight_forwarding
```

## Requirements

- Frappe Framework: v15.x
- ERPNext: version-15
- Python: 3.10.x

## Features

- Division-first architecture
- Dynamic naming series
- Custom masters (FF Port, FF Airport)
- Rate Management
- Employee Advance with multi-item allocation
- Expense Claim consumption tracking
